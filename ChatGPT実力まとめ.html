2026年 一気に進化しました
1
集合写真をイラスト化 → 名前入り・顔も補完してくれた
オフ会の写真を投げて「イラスト化して、左から順に名前を○○・○○・○○」と指示するだけで…
  • 写真をおしゃれなイラストに変換
  • 名前を吹き出し付きで自動配置
  • 顔が隠れていた人も、顔が見えるように補完してくれた
具体例① ／ 4月6日 京都オフ会
元の写真（IMG_1672） 3人の笑顔の記念写真
→
ChatGPTが生成したイラスト 水彩画風のイラストに変換され、みんな若返っています（笑）
✨ ポイント：名前を楽しそうな雰囲気の吹き出しで入れてくれた
具体例② ／ 4月26日 京都オフィス（Claude Code勉強会）
元の写真（IMG_1949） 右から2人目と3人目の顔が重なっていて、顔が見えない
→
ChatGPTが生成したイラスト 水彩風イラストに変換され、全員の顔がはっきり見えるように
✨ ポイント：顔が重なって見えなかった2人も、ちゃんと顔が見えるように補完！
具体例③ ／ 4月26日 京都オフィス（Claude Code 初操作つまずきラボ）
元の写真（IMG_1945） 9人の集合写真
→
ChatGPTが生成したイラスト 全9名の名前＋一言コメントが自動生成された
✨ ポイント：名前を渡したら「明るくて元気！」「クールな発想で頭の回転が速い」など、一人ひとりへの一言コメントを自動でつけてくれた！

ただし、とくじい・やっさんの位置が入れ替わっていた（名前から判断して、名前の場所を入れ替えてくれたのかも・・・よけいなお世話だけど）
2
机の上のモノを全部消して、ジモティ出品用の写真に変換
デスクをジモティに出品したいけど、PC・書類・ペットボトルを片付けるのが面倒… そのまま写真を撮って「机の上のものを削除して」と指示するだけで：
  • キーボード・モニター・書類・ケーブル類がすべて消える
  • 背景も自然なオフィス風に整えられた
  • まるでスタジオで撮影したような商品写真に仕上がった
元の写真（IMG_1917） モニター・キーボード・書類・ペットボトル・カバンが散乱した状態
→
ChatGPTが生成した画像 机の上が完全にスッキリ。背景も自然なオフィス調に
✨ ポイント：片付けゼロで出品用の商品写真が完成！これはジモティ出品に使えた！
3
写真1枚からLINEスタンプ16種を作成 → クリエーターズマーケット申請用にすべて準備してくれた
写真を1枚渡して「これはかよさんの写真です。この写真の顔をイラスト化してLINEスタンプを作って下さい」と指示するだけで…
  • 16種類のスタンプ画像（「こんにちは！」「いいね！」「ありがとう」など各種表情＋文字入り）を生成
  • メイン画像・タブ画像・スタンプ画像の3種類を規定サイズで出力
  • すべて透過PNG形式でクリエーターズマーケットにそのままアップできる状態に仕上げてくれた
元の写真（1枚だけ渡した） 「この写真からLINEスタンプを作って」と指示しただけ
→
ChatGPTが生成したLINEスタンプ（16種） 各種表情＋ひとことメッセージ入りのスタンプが一気に完成
✨ ポイント：メイン画像（240×240px）・タブ画像（96×74px）・スタンプ画像16枚（370×320px）をすべて透過PNGで出力。クリエーターズマーケットの申請規格にぴったり合わせてくれた！
